Policies validation API

API to validate one or more policies for a document

Validate a document

Validate a document with a single policy, multiple policies, all the policies or the policies in a suite

If neither the optional parameters "id" or "suite" will be used the documentwill be validated through all the policies sotred on OPA Server

query Parameters
id
Array of strings

Optional - identifier/s of the policy or the policies to use in the validation of the document

suite
Array of strings

Optional - identifier/s of the suite or the suites to use in the validation of the document

Request Body schema: application/json
required
object

JSON object of the document to be validated

Responses

Generated server url

/api/v1/up/policy-service/validate
